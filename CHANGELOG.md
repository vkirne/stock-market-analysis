# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-16

### Added
- Initial release of Stock Market Analytics Dashboard
- Real-time stock data visualization from Alpha Vantage API
- Support for major stocks (IBM, AAPL, MSFT, GOOGL, AMZN, TSLA, META)
- Multiple time intervals (1min, 5min, 15min, 30min, 60min)
- Technical indicators:
  - RSI (Relative Strength Index)
  - MACD (Moving Average Convergence Divergence)
  - Bollinger Bands
  - Moving Averages (5, 20, 50, 200 periods)
- Watchlist functionality for favorite stocks
- Auto-refresh feature with configurable intervals
- Advanced interactive charts (Plotly)
- BUY/SELL/HOLD trading signals
- Glass morphism UI design with purple/yellow/orange theme
- Docker containerization
- Docker Compose orchestration
- Educational disclaimer modal
- Toast notifications
- Price change highlighting
- Keyboard shortcuts documentation

### Features
- Session-based data caching
- Error handling for API failures
- Responsive layout
- Health checks in Docker
- Environment variable configuration

### Documentation
- Comprehensive README
