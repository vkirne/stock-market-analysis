# Product Overview

Stock Market Analytics Dashboard is an educational Python application for real-time stock market data visualization and technical analysis.

## Purpose

This is a **demonstration and educational tool only** - not for real trading or investment decisions. The application showcases technical analysis features, data visualization, and modern UI patterns.

## Key Features

- Real-time stock data from Alpha Vantage API
- Support for major stocks: IBM, AAPL, MSFT, GOOGL, AMZN, TSLA, META
- Multiple time intervals: 1min, 5min, 15min, 30min, 60min
- Technical indicators: RSI, MACD, Bollinger Bands, Moving Averages
- Interactive charts: candlestick, price trends, volume analysis
- Watchlist management (up to 20 stocks)
- Auto-refresh functionality with configurable intervals
- Modern UI with glass morphism effects and purple/yellow/orange theme

## Important Constraints

- **Educational purpose only** - always include disclaimer warnings
- API rate limits: 5 requests/minute on free tier
- Data may be delayed or inaccurate
- No financial advice provided or implied
